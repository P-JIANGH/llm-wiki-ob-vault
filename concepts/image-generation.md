# Image Generation

AI 图像生成技术，是 AIGC（AI Generated Content）的重要组成部分。

## 技术路线

### Diffusion 模型

- [[stable-diffusion]] 系列
- DALL-E 3
- Midjourney
- Imagen (Google)

### 自回归模型

-Parti (Google)
- Emu (Meta)

## 游戏开发应用

- **概念美术**：快速生成游戏场景、角色概念图
- **纹理生成**：基于 prompt 生成贴图
- **UI 素材**：按钮、图标、背景
- **NPC 立绘**：角色形象快速原型
