# Manus

[[autonomous-agents]] | [[ai-agent-platforms]]

## Overview

Manus is a general-purpose AI agent that can autonomously complete complex tasks across diverse domains. It uses a multi-agent architecture where specialized sub-agents handle different aspects of a task.

## Key Features

- **Multi-agent orchestration**: Multiple specialized agents working together
- **Memory persistence**: Long-term memory across sessions
- **Tool use**: Web search, code execution, file manipulation
- **Domain generality**: Works across coding, research, creative tasks

## Relationship to Other Projects

- Similar in scope to [[AutoGPT]] and [[babyagi]]
- Related to [[AgentGPT]] as cloud-hosted autonomous agents
