# Multi Agent Systems

多智能体系统，多个 AI Agent 协作完成复杂任务的架构模式。

## 核心范式

- **协作**：多个 Agent 分工，共同解决问题
- **竞争**：Agent 间对抗（如游戏 AI）
- **层次**：上级 Agent 调度下级 Agent
