# Qwen2.5

[[open-source-llm-projects]] | [[foundation-models]]

## Overview

Qwen2.5 is Alibaba DAMO Academy's flagship open-source LLM series. It covers 0.5B to 72B parameters and includes specialized variants for coding (Coder) and mathematics.

## Key Variants

- **Qwen2.5**: Base models 0.5B–72B with improved reasoning
- **Qwen2.5-Coder**: Finetuned for code generation
- **Qwen2.5-Math**: Specialized for mathematical reasoning
- **Qwen2.5-72B-Instruct**: Top-tier instruction-following model

## Relationship to Other Projects

- Competitor to [[Meta-LLaMA]] and [[Mistral]] in the open LLM space
- Used in [[Qwen-Agent]] and [[vLLM]] deployments
