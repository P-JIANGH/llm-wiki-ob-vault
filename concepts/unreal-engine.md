# Unreal Engine

[[game-development]] | [[3d-engines]]

## Overview

Unreal Engine is Epic Games' flagship game engine, widely used in AAA game development and real-time visualization. Version 5 introduced Nanite (virtualized geometry) and Lumen (全局光照).

## Key Technologies

- **Nanite**: Virtualized micropolygon geometry system
- **Lumen**: Dynamic global illumination system
- **MetaHuman**: Photorealistic digital human creation
- **Unreal Engine 5**: Free to use with 5% royalty after $1M revenue

## Relationship to Other Projects

- Integration with [[Llama]] via [[unreal-engine-5-llama-lora]] for in-game AI
- Compared with [[Unity]] and [[Godot]] as game engine alternatives
- [[UnrealGPT]] provides LLM integration for Unreal Engine
