# Persistent Memory System

## Definition
AI 角色的长期记忆存储系统，角色能记住历史对话和事件，形成连续的生活体验。

## Key Features
- 持久化存储（JSON 格式）
- 记忆检索和查询
- 基于记忆的个性化行为发展
- 格式化存储结构
