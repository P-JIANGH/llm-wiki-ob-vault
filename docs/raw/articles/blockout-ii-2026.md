## Official Description
BlockOut II is an OpenGL adaptation of the original BlockOut DOS game edited by California Dreams in 1989.
BlockOut II has the same features as the original game with few graphic improvements.
Score calculation is also nearly similar to the original Game.
BlockOut II has been designed by an addicted player for addicted players.

## Dependencies (Linux)
