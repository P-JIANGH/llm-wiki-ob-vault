# AI Town

a16z 推出的多 Agent 虚拟小镇模拟项目。基于 [[generative-agents]] 思想，展示多个 AI 角色如何在虚拟世界中自主社交、对话、行动。

## 技术栈

- **前端**：React + TypeScript
- **后端**：Convex（实时数据库）
- **AI**：接入 LLM API
- **部署**：可本地运行

## 核心特性

- 多 Agent 交互模拟
- 可自定义 Agent 角色
- 实时状态更新

## 游戏开发参考

AI Town 是 [[generative-agents]] 的工程化落地，其多 Agent 协作模式可应用于：
- NPC 社会模拟
- 开放世界 NPC 行为
- 游戏内经济系统模拟
