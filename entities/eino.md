# Eino

Go 语言的多 Agent 编排框架，专注于构建 LLM 驱动的应用。

## 特点

- **Go 原生**：充分利用 Go 的并发优势
- **轻量**：无过多抽象层
- **可扩展**：易于集成自定义组件

## 与其他框架对比

| 框架 | 语言 | 特点 |
|------|------|------|
| LangChain | Python | 最成熟，文档丰富 |
| LlamaIndex | Python | RAG 强项 |
| Eino | Go | 高并发优势 |
| LangGraph | Python | 状态机建模 |

## 游戏开发应用

Go 的高并发特性适合游戏服务器场景，Eino 可用于：
- 游戏内 AI 对话服务
- NPC 行为树编排
- 实时策略 AI
