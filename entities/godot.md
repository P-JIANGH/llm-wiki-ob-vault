# Godot

开源 2D/3D 游戏引擎，MIT 许可证。

## 特点

- **开源免费**：MIT 许可证，无版税
- **2D 原生**：内置 2D 引擎
- **轻量**：对硬件要求低
- **GDScript**：类 Python 脚本语言

## 游戏类型

- 2D 平台游戏
- 3D 冒险游戏
- RPG
