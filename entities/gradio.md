# Gradio

Python Web UI 框架，专为机器学习模型设计。

## 特点

- 简单易用，Python 装饰器风格
- 内置输入输出组件（文本/图像/音频/视频）
- 支持 Chatbot 界面
- 一键部署到 HuggingFace Spaces

## 游戏开发应用

- AI 对话机器人 demo
- 游戏 AI 演示界面
- 模型推理测试
