# HKUDS

**Hong Kong University Data Science Lab** — 香港大学数据科学实验室。

## 开源项目

| 项目 | 描述 | 关系 |
|------|------|------|
| [[nanobot]] | 极轻量个人 AI Agent 框架（~4000行） | 核心项目 |
| [[openclaw]] | AI Agent 框架（设计灵感来源） | 参考 |
| [[clawteam]] | Agent Swarm 编排框架 | 核心项目 |
| CLI-Anything | 让所有软件 Agent 化 | 姐妹项目 |

## 定位

HKUDS 专注于 **AI Agent 基础设施**：让单个 Agent 更稳定（nanobot）、让多个 Agent 协作更容易（ClawTeam）。核心理念：极简代码 + 零外部依赖 + 任意 CLI agent 兼容。
