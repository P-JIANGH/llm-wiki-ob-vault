# IP-Adapter

[[stable-diffusion]] 的图像提示适配器，通过额外编码器将参考图像融入生成过程。

## 功能

- **风格迁移**：保持构图，用参考图的风格生成
- **人物一致性**：保持角色特征
- **产品图生成**：用参考商品生成新场景

## 游戏开发应用

- 角色设计：保持角色一致性生成不同姿态
- 场景风格统一
- UI 素材生成
