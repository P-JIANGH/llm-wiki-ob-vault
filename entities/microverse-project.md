# Microverse

## Overview
基于 Godot 4 开发的多智能体 AI 社交模拟系统沙盒游戏。AI 角色拥有独立思维和记忆，能自主社交、完成任务、发展社会关系。类似斯坦福 AI 小镇。

## Key Facts
- **开发者**: KsanaDock (时空码头)
- **GitHub**: github.com/KsanaDock/Microverse
- **原始仓库**: gitcode.com/gh_mirrors/micr/Microverse (GitHub 镜像)
- **引擎**: Godot 4.3+
- **语言**: GDScript
- **许可证**: MIT
- **Steam 版本**: Microverse In Box (即将上线)

## Features
- 8 个预设 AI 角色（Alice/Grace/Jack/Joe/Lea/Monica/Stephen/Tom）
- 持久化长期记忆系统
- 自主任务管理
- 多 AI 服务集成（OpenAI/Claude/Gemini/DeepSeek/豆包/Kimi/Ollama）
- 办公室场景地图
- 存档/读档系统
