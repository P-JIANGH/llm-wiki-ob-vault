# OpenClaw

开源 AI Agent 框架，[[nanobot]] 的设计灵感来源，[[hkuds]] 的参考架构。

## 概述

OpenClaw 是 Anthropic 早期开源的个人 AI Agent 项目，核心理念：用极简代码实现稳定的、长期运行的 AI Agent。

## 与 nanobot 的关系

- nanobot 公开描述为"灵感来自 OpenClaw"
- nanobot 比 OpenClaw 代码量减少 99%（~4000 行 vs ~430,000 行）
- nanobot 复刻了 OpenClaw 的核心思想并大幅精简实现
