## Overview

Dora AI is a generative AI website builder by [Dora](https://dora.run/) that allows users to create websites from natural language prompts. It is marketed as "Sites beyond imagination, one prompt away."

## Key Features

- **Text-to-website generation**: Users describe a website concept in natural language, and Dora AI generates a full responsive site.
- **AI Pipeline**: The tool follows a multi-step pipeline:
  1. Analyzing prompt — determines subject and style
  2. Crafting designs — generates original images and copy within a responsive layout
  3. Tweak, iterate, publish — user refines the generated result
- **Brand-aware**: Supports building sites with specific brand identities (Nike, Tesla, Chanel, Lego, McDonald's, Netflix examples shown)
- **100% prompt-relevant**: Claims to use no templates or stock photos — all content generated from the prompt
- **Multi-style support**: Can generate sites in any topic and style
- Won a ProductHunt Golden Kitty award
