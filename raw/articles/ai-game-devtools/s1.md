# s1: Simple Test-Time Scaling

## 基本信息
- **GitHub**: https://github.com/simplescaling/s1
- **论文**: https://arxiv.org/abs/2501.19393
- **模型**: https://hf.co/simplescaling/s1.1-32B
- **数据**: https://hf.co/datasets/simplescaling/s1K-1.1

## 项目概述
s1 是一个极简的测试时扩展（test-time scaling）方案，仅用 1,000 个样本 + budget forcing 即实现了与 o1-preview 相当的推理能力。

## 核心贡献
1. **s1K 数据集**：59,000+ 样本，经过 Gemini 和 Qwen 生成的 reasoning traces
2. **s1.1 模型**：基于 Qwen2.5-32B 微调，通过 r1 推理 traces 进一步提升
3. **Budget Forcing**：强制模型延长推理 token 数，通过忽略 end-of-thinking token 让模型继续思考

### 依赖
- PyTorch 2.5.1, transformers 4.46.1, vLLM 0.6.4
- TRL (Transformers Reinforcement Learning) 0.12.0
- WandB 用于训练监控

### 推理方式
- **vLLM**：推荐，支持 tensor_parallel_size=2
- **Transformers**：直接加载 HuggingFace 模型
- **Budget Forcing**：在推理时强制 token 预算，超时后忽略 stop token 强制继续思考

### 训练
- 推荐 16 H100 (2节点×8卡)
- s1.1 使用 block_size=20000 避免 OOM
- 支持 FSDP (Fully Shard Data Parallel)
