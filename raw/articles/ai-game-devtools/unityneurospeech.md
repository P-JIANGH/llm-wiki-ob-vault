# UnityNeuroSpeech

> Source: https://github.com/HardCodeDev777/UnityNeuroSpeech
> Extracted: 2026-04-22 (web extract; GitHub/gitcode/gitee clone all failed)

**UnityNeuroSpeech** is an open-source Unity framework for creating fully voice-interactive AI agents that run entirely offline. Billed as "the world's first game framework that lets you talk to AI in real time — locally," it supports any custom voice with no cloud dependencies.

**Core promise:** All locally. All offline. No subscriptions, no accounts, no OpenAI API keys.

## Core Features

| Feature | Implementation / Detail |
|---|---|
| 🎙️ **Voice Input** | Speech-to-text via [whisper.unity](https://github.com/Macoron/whisper.unity) |
| 🧠 **AI Brain (LLM)** | Local models via [Ollama](https://ollama.com) |
| 🗣️ **Custom TTS** | Any voice via [Coqui XTTS](https://github.com/idiap/coqui-ai-TTS) |
| 😄 **Emotions** | Auto-parsed emotion tags from LLM output |
| 🎬 **Actions** | Auto-parsed action tags from LLM output |
| 🎛️ **Agent API** | Event subscription (e.g., `BeforeTTS()`) |
| 📝 **History Saving** | JSON dialog history with optional AES encryption |
| 🛠️ **Editor Tools** | Create, manage, and customize agents inside Unity Editor |
| 🌐 **Multilingual** | 15+ languages (English, Russian, Chinese, etc.) |
| 🔊 **Multi-Agent Voice Support** | Each agent can use distinct voice files per language |
| ⚡ **Performance** | Async via UniTask (replaces Coroutines/Tasks) |
| 🔧 **Build Support** | Full compatibility with Mono and IL2CPP |

## Emotion & Action Tag Examples

The framework automatically parses special tags generated by the LLM:

**Emotions:** `<happy>`, `<sad>`
**Actions:** `<turn_off_lights>`, `<play_cutscene_123>`
**Agent API Event:** `BeforeTTS()`
