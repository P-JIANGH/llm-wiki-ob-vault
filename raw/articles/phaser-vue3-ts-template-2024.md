# Phaser 3 + Vue 3 TypeScript Official Template

Source: https://phaser.io/news/2024/02/official-phaser-3-and-vue-3-ts-template
Date: 2026-04-22

## Overview
Phaser Studio official template for Phaser 3 + Vue 3 + TypeScript + Vite.
Includes a bridge for Vue to Phaser game communication.

## Benefits
- Vue's reactivity system + TypeScript static typing
- Enhanced maintainability and scalability
- Smooth integration of game mechanics with UI
- Official support from Phaser Studio
